Project Description

A game in which you assume the role of a viking trapped in a tower filled with weird monster. You are now trying to escape from it.

Game Mechanics

Use the WASD keys for movement, Space for attack. You can maneuver around with climbable flag.

Showcase

Here is a project I created using the course "The Ultimate Guide to C# Unity 2D Programming (2024)" by Michael Murr on Skillshare. This project helped me practice 2D game development as I prepare for my upcoming participation in the 2D Jam.

Enemy Script 

        void Update()
        {
            if (isLookingLeft())
                rb.velocity = new Vector2(-enemySpeed, 0f);
            else
                rb.velocity = new Vector2(enemySpeed, 0f);
        }
    
        public void Die()
        {
            anim.SetTrigger("Die");
            GetComponent().enabled = false;
            GetComponent().enabled = false;
            rb.bodyType = RigidbodyType2D.Static;
            StartCoroutine(AfterDyingProcedure());
        }
    
        IEnumerator AfterDyingProcedure()
        {
            yield return new WaitForSeconds(2);
            Destroy(gameObject);
        }
    
        private void OnTriggerExit2D(Collider2D collision)
        {
            Vector3 scale = transform.localScale;
    
            transform.localScale = new Vector2(-scale.x, scale.y);
        }
    
        private bool isLookingLeft()
        {
            return transform.localScale.x > 0f; 
        }
